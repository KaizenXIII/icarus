# Icarus

A space strategy game built in Unity — currently prototyping *Admirals of the Void*, an isometric pixel art grand-strategy experience set in a procedurally generated galaxy. You command a faction's fleet across a living, zoomable galaxy map spanning five nested tiers of detail.

---

## Project Status

Early prototype. The following are working:

- Procedural galaxy generation with spiral arms, hyperlane networks, and faction territory
- Five-tier seamless zoom system (Galaxy > Sector > System > Fleet > Ship)
- Turn-based phase structure (War Council > Command > Resolution)
- Three-faction setup with resource collection per turn
- Fleet controller with admiral/hero data and ship composition
- Interactive React canvas prototype (`admirals-zoom-prototype.jsx`) used to validate zoom feel and pixel-art rendering before Unity implementation

---

## Concept

**Genre:** Turn-based space grand strategy
**Working title:** Admirals of the Void
**Engine:** Unity 6000.0.54f1
**Art style:** Pixel art, dark palette, neon accents

Three factions — The Sovereignty, The Covenant, and The Syndicate — compete for control of a procedurally generated galaxy.

## Features

### Galaxy Map (Tier 1-2)
- Procedurally placed star systems along logarithmic spiral arms
- Hyperlane network auto-generated between systems
- Faction territory overlay colors systems and lanes by owner

### Zoom System (5 Tiers)

| Tier | Zoom Size | Content Revealed |
|---|---|---|
| 1 — Galaxy | > 50 | Full galaxy, faction labels, hyperlanes |
| 2 — Sector | 22-50 | System cluster, population labels |
| 3 — System | 10-22 | Individual star, orbiting planets |
| 4 — Fleet | 4-10 | Fleet formation, ship silhouettes, roster panel |
| 5 — Ship | < 4 | Individual ship stats, admiral portrait, modules |

### Turn System
Three-phase turns: War Council > Command > Resolution

### Factions & Heroes
- Three factions with capital systems and starting territories
- Resources: Credits, Fuel, Materials, Influence
- Budget allocation across Fleet, Tech, Espionage, Colony, Diplomacy
- `HeroData` with five stats and named abilities

### Ship Classes
Corvette, Destroyer, Cruiser, Carrier, Dreadnought, Support — each with hull, firepower, speed, module slots, and build cost.

## Getting Started

1. Clone the repository.
2. Open **Unity Hub**, click **Open**, and select the `unity/` folder.
3. In the Unity Editor, run: **Menu bar > Admirals > Bootstrap Galaxy Scene**
4. Open `GalaxyScene` and enter Play mode.

### Controls

| Input | Action |
|---|---|
| Scroll wheel | Zoom toward cursor |
| Left/Middle mouse drag | Pan camera |
| Arrow keys / WASD | Pan camera |
| `1` - `5` | Jump to zoom tier |
| Double-click system | Fly-to zoom |
